Privacy Policy

1. Data Collection

myTuur GmbH collects data to ensure the functionality of the app and to improve the user experience. This data includes:

  • Personal data (e.g., name, email address)
  • Location data
  • Usage data (e.g., search queries, bookings)

2. Use of Data

The collected data is used to:

  • Enable the booking and execution of guides.
  • Improve and customize the app.
  • Provide support and respond to inquiries.

3. Data Sharing

Data will not be shared with third parties without the user's consent unless legally required or necessary to provide the app's service.

4. Data Security

myTuur GmbH takes appropriate measures to ensure the security of user data and protect it from unauthorized access.

5. User Rights

Users have the right to access their stored data, request correction or deletion of their data, and object to data processing.
